据新华社消息，中央人民政府决定，任命刘光源为外交部驻香港特别行政区特派员公署特派员，免去谢锋的外交部驻香港特别行政区特派员公署特派员职务。刘光源此前担任中国驻波兰大使，谢锋已于1月13日奉调离任回京，后出任外交部副部长，分管政策规划、美大和拉美地区、涉外安全事务以及翻译工作。长安街知事（微信ID：Capitalnews）注意到，刘光源有着丰富的驻外经历，曾多次驻非洲、驻美。在波兰任职期间，刘光源与美国驻波兰大使展开多轮交锋，一一回击其抹黑言论。刘光源是河南省淮阳县人，1986年从郑州大学毕业后，进入外交部工作，先后在外交部非洲司、驻加纳大使馆、驻尼日利亚大使馆、外交部干部司任职。 2000年，刘光源赴美履职，出任驻旧金山总领馆副总领事，两年后回京任外交部干部司副司长，2007年再次驻美，任驻美大使馆公使。2010年，刘光源出任驻肯尼亚大使，兼任常驻联合国环境规划署代表、常驻联合国人类住区规划署代表，4年后，刘光源回京任外交部涉外安全事务司司长。值得一提的是，涉外安全事务司的主要职责包括研究涉及国家安全问题的涉外事宜，协调境外非政府组织在华活动管理工作等。2018年3月，刘光源履新驻波兰大使。长安街知事（微信ID：Capitalnews）注意到，针对时任美国驻波兰大使莫斯巴赫多次通过社交媒体就新冠疫情、5G等问题抹黑中国的情况，刘光源利用各种事实数据逐一回击。去年3月，莫斯巴赫在波最大门户网站之一“ONET”就疫情问题发文污蔑中国，还利用推特进一步传播。次日，刘光源大使在同一网站发表题为“美国歪曲事实，为防疫不力甩锅”的署名文章，并在推特上@美大使，提醒其阅读。去年7月，莫斯巴赫在推特上转发美国务院官员文章，称中国5G企业“不可信”，之后接连发表推文，污蔑中企被中国政府利用进行监控等。刘光源随即发表17条推文，揭批美方毫无证据地滥用国家力量打压中企，并在波兰媒体上发表署名文章。
